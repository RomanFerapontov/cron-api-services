## Start

- install Node.js (from version 14)
- install NPM
- install PosgreSQL

### Installation

1. Clone repository:

```bash
git clone git@github.com:RomanFerapontov/cron-api-services.git
```

2. Change directory:

```bash
cd name_project_directory
```

3. Install dependencies:

```bash
npm install
```
